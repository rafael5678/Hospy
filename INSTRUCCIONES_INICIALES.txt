📋 INSTRUCCIONES PARA EMPEZAR (3 Pasos Simples):

┌─────────────────────────────────────────────────────────────┐
│  PASO 1: Crear archivo .env.local                           │
└─────────────────────────────────────────────────────────────┘

1. En la carpeta raíz del proyecto, crea un archivo llamado:
   .env.local

2. Copia y pega este contenido:

   MONGODB_URI=mongodb://localhost:27017/hospy
   NEXT_PUBLIC_APP_URL=http://localhost:3000

3. Guarda el archivo


┌─────────────────────────────────────────────────────────────┐
│  PASO 2: Instalar MongoDB                                   │
└─────────────────────────────────────────────────────────────┘

Opción A: MongoDB Local
→ Descarga: https://www.mongodb.com/try/download/community
→ Instala MongoDB Community Server
→ Instala MongoDB Compass (interfaz visual)

Opción B: MongoDB Atlas (Gratis en la nube)
→ Crea cuenta: https://www.mongodb.com/cloud/atlas
→ Crea un cluster gratuito
→ Copia tu cadena de conexión a .env.local


┌─────────────────────────────────────────────────────────────┐
│  PASO 3: Ejecutar el proyecto                               │
└─────────────────────────────────────────────────────────────┘

Abre tu terminal en esta carpeta y ejecuta:

   npm install
   npm run dev

Luego abre en tu navegador:
   http://localhost:3000
